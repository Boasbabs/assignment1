Assignment 1

The challenges of first assignment.


  1. Hello World to the screen.
  2. Program that asks the user for her name and greets her with her name.
  3. Program such that only the users Alice and Bob are greeted with their names.
  4. Takes a number n as argument and prints the sum of the numbers 1 to n.
  5. Function such that only multiples of three or five, but not both are considered in the sum.
  6. Function that takes a number 'n' as argument and prints a multiplication table for 'n' up to 12.
  7. Function that takes a four digit year number as input and prints the next 100 leap years from the given year
